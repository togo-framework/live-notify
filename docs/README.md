# live-notify — docs

Bridge live agent replies to the togo notifications system (Slack, Discord, web-push, FCM, Pusher, mail).

- **Install:** `togo install togo-framework/live-notify`
- **Marketplace:** https://to-go.dev/plugins/live-notify
- **Source:** https://github.com/togo-framework/live-notify

See [usage.md](usage.md) for wiring channels.
